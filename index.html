Programming languages

This is my first major project where i have had to go off and learn how to write the code myself. for this project i used three different languages: html, css, and javascript.


HTML

I found all these relatively easy to use for the basic functions required for this project. I have used html to create the bones of the website ordering all the elements and putting in each section. for example the pictures, paragraphs, buttons, etc. This allowed me to input all the functionality of the website and actually make it a thing.





CSS

Secondly I have used CSS which allows me to format the website and decide how it all looks. This controls everything from background colour to shape and size of images. I decided to go for a dark navy blue background with a very light grey navigation bar at the top, two colours I believe complement each other quite well on this site. Additionally CSS allows me to move around all the elements of the website which lets me put the paragraphs and pictures wherever I want them.





Javascript

Finally I have used javascript to allow me to incorporate some interactivity to the website. This language controls anything that the user interacts with on the website, like buttons or a personalised welcome message. I found this the hardest language out of the three as it seemed to be the most complicated and hardest to include with the rest of the website.
